# Quick Start Guide

## Установка и запуск

1. **Установите зависимости:**
```bash
cd frontend-experimental-react
npm install
```

2. **Запустите dev сервер:**
```bash
npm run dev
```

3. **Убедитесь, что бэкенд запущен на порту 8080**

4. **Откройте браузер:**
```
http://localhost:3000
```

## Доступные страницы

- `/login` - Вход
- `/register` - Регистрация
- `/quests` - Квесты (Available/Active/Completed)
- `/quests/:id/details` - Детали квеста
- `/shop` - Магазин квестов
- `/friends` - Друзья
- `/profile` - Профиль пользователя
